# paper_scanner_example

Demonstrates [`paper_scanner`](../packages/paper_scanner): launching the
scanner with the default theme and with a heavily customized theme, then
showing the resulting page images and PDF path.

## Run

The platform runner projects (`android/`, `ios/`) are generated by Flutter and
are not checked in. Generate them once, then run:

```bash
cd example
flutter create --platforms=android,ios .
flutter pub get
flutter run        # use a REAL device — the camera is required
```

## Required permissions

After generating the runners, add the camera usage strings.

**iOS** — `ios/Runner/Info.plist`:

```xml
<key>NSCameraUsageDescription</key>
<string>Scan documents with the camera.</string>
```

**Android** — `android/app/src/main/AndroidManifest.xml`:

```xml
<uses-permission android:name="android.permission.CAMERA" />
```

Also ensure `minSdkVersion 21` (or higher) in
`android/app/build.gradle`.
